# Interactables
```
{
  "state": "STATE"
}
```

Depending on the _type_ of Interactable, the `STATE` is different:

## JumpPad States
Does not need to be specified.

## Switch States
Does not need to be specified.

## Door States
- `Open`
- `Closed`
- `Opening`
- `Closing`
